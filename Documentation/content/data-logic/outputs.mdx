# Outputs
Rolewise produces structured interpretation, not verdicts.
## Output types
Common outputs include:
- role summaries
- structured role traits (Role DNA)
- blockers and unknowns
- trade-offs
- decision records
- reflection views
## Output behaviour
Outputs should:
- clarify the role
- highlight what matters
- surface uncertainty

They should not:
- collapse complexity into a score
- present a final answer
- imply certainty where none exists
## Output rule
Every output should help the user think more clearly.
## In one sentence
Rolewise outputs clarify, they do not decide.
