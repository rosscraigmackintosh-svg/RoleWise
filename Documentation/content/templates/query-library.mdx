# Query library
This defines the kinds of questions you can ask your data.
## Pipeline
- How many roles are active?
- Where are roles dropping off?
## Market
- What types of roles am I seeing most?
- Which industries appear most often?
## Response
- How long are responses taking?
- Which roles get replies?
## Risk
- Which roles had similar blockers?
- Where am I seeing repeated issues?
## Strategy
- What should I do next?
- What should I stop doing?
## In one sentence
Good questions matter more than more data.
