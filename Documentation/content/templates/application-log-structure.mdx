# Application log structure
Each role should have a consistent structure.
## Core fields
- Company
- Role title
- Source (job board, recruiter, referral)
- Date seen
## Role details
- Location
- Work model
- Salary (if stated)
- Industry
- Company stage
## Decision
- Applied / Skipped / Watching
- Date of decision
## Process
- Stage reached
- Interview steps
- Outcome
## Signals
- Response timing
- Risk flags
- Notes
## In one sentence
Consistency enables pattern recognition.
