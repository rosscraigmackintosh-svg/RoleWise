# Spacing and radius
Spacing defines rhythm.
## Spacing
Spacing should:
- separate clearly
- create breathing room
- group related content
## Radius
Radius should be subtle.
Avoid overly soft or playful shapes.
## In one sentence
Spacing creates calm. Radius stays controlled.
