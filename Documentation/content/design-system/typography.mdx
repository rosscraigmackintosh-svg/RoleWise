# Typography
Typography does most of the work.
## Principles
- hierarchy should be obvious
- body text should be easy to read
- headings should feel firm, not loud
- labels should guide structure
## Practical rules
- strong contrast between headings and body
- generous line height
- avoid small text in core content
- use uppercase labels sparingly
## In one sentence
Typography should carry clarity without needing decoration.
