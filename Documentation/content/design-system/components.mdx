# Components
Components should feel simple and dependable.
## Principles
- clear before clever
- consistent before expressive
- quiet before attention-seeking
## Behaviour
Components should:
- support interpretation
- support comparison
- support reflection

They should not dominate attention.
## In one sentence
Components should feel like tools, not features.
