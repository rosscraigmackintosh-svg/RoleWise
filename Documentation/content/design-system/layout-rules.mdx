# Layout rules
Layout should support thought, not speed.
## Principles
### Strong frame
The interface should feel structured and stable.
### Readable width
Content should not stretch too wide. Reading should feel comfortable.
### Clear separation
Navigation, content, and supporting elements should be distinct.
### Low noise
Use spacing and borders more than colour.
## In one sentence
Layout should feel like a calm document, not a busy interface.
