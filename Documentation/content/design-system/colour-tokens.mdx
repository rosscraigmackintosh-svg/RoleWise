# Colour tokens
Colour should be restrained.
## Role of colour
Use colour for:
- contrast
- hierarchy
- subtle state signals
- structure

Not for excitement.
## Palette behaviour
- mostly neutral
- one restrained accent
- soft contrast
- warm background tones
## In one sentence
Colour should support clarity, not create noise.
