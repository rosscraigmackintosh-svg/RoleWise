# How to use Rolewise
Rolewise fits into how you already work. You don't need to change your process — just make it clearer.
## Basic flow
A typical flow looks like this:
1. Find a role (job board, recruiter, referral)
2. Paste the job description
3. Review the analysis
4. Decide: apply, skip, or watch
5. Track what happens
6. Review patterns over time

## When to use it
Use Rolewise when:
- a role looks interesting but unclear
- you are unsure whether to apply
- you want to avoid low-quality applications
- you want to understand patterns over time

## When not to use it
You do not need Rolewise for:
- obvious decisions
- roles you already fully understand
- quick exploratory browsing

## What good use looks like
Good use of Rolewise is:
- selective
- thoughtful
- consistent over time

It is not about analysing every role.
## Common mistake
Using Rolewise as a scoring tool. It helps you think, not what to do.
## In one sentence
Use Rolewise to slow down decisions, not to speed them up.
