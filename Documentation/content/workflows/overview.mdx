# Workflows
Rolewise is a system of flows, not screens.
## Why workflows matter
The product only works when the pieces connect:
- discovery
- analysis
- decision
- tracking
- reflection
## Core loop

Discover → Analyse → Decide → Track → Reflect

## Product goal
Make this loop:
- simple
- consistent
- repeatable
## In one sentence
Workflows turn Rolewise into a system, not a tool.
