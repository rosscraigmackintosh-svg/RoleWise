# Documentation updates
This page tracks meaningful changes to documentation.
## Record updates when
- a new page is added
- a section meaningfully changes
- structure is updated
- terminology shifts
- interpretation changes
## Do not record
- small copy edits
- formatting fixes
- minor clarity tweaks
## Format
Each entry should include:
- date
- change summary
- reason for change
## Example
- 2026-03 — Added Data & Logic section to clarify system behaviour
## In one sentence
Record changes when they affect meaning or structure.
