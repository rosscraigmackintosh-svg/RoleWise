# Changelog
The changelog records meaningful changes to the Rolewise manual.
## Purpose
It exists to:
- make changes visible
- preserve reasoning behind updates
- track how the system evolves
## What gets recorded
- documentation changes
- rule changes
- structural changes
- product thinking shifts
## What does not
- minor copy edits
- formatting tweaks
- small wording improvements
## In one sentence
The changelog tracks changes that affect understanding, not presentation.
